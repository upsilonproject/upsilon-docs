== Services & Introduction

=== Architecture Overview

* xref:cli.adoc[CLI] - The command line interface.
* xref:drone.adoc[Drone] - The drone is the main worker, it run service checks and collects results.
* xref:web.adoc[Web] - The web interface provides a graphical dashboards, and administrator command and control.
* MobileWeb - A native Android application with shortcuts for the web application, as as the ability to recieve notifications from reactors.
* xref:reactor.adoc[Reactor] - Alerting and notifications.
* xref:custodian.adoc[Custodian] - Provides an API on top of the database.
