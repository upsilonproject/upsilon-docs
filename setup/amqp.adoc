=== AMQP Server

The AMQP server provides a message queue, that allows services to talk to one another. The AMQP server supported and tested by the upsilon project is rabbitmq. 

To install rabbitmq on a fedora based server:

....
user@root: dnf install rabbitmq-server -y
user@host: systemctl enable --now rabbitmq-server
....
